# Elite Mode Studio

> Autonomous social media content engine — single posts and multi-slide carousels powered by NVIDIA NIM LLMs, Tavily web search, and a Fabric.js Design Studio.

Built as an Electron desktop app. Posts generated by this engine have hit 149K, 152K, and 1.3M views on Instagram.

---

## What it does

1. **Fetches verified news** from 25+ Tier-1 sources (Reuters, AP, Bloomberg, FT, BBC, etc.) using Tavily
2. **Generates social posts** using NVIDIA's Llama 3.3 70B — title, caption, image prompts, highlight words, sources
3. **Designs carousels** in a full Fabric.js Design Studio with rulers, guides, snapping, and frames
4. **Exports** as PNG batches via native file dialogs

---

## Tech Stack

| Layer | Tech |
|-------|------|
| Desktop shell | Electron 29 |
| Frontend | React 18 + Vite 5 + Tailwind CSS |
| Backend | Python FastAPI on `127.0.0.1:8000` |
| AI / LLM | NVIDIA NIM — Llama 3.3 70B |
| Embeddings | NVIDIA NV EmbedQA 1B v2 |
| Reranker | NVIDIA Nemotron Rerank 1B v2 |
| Vector store | FAISS (in-memory) |
| Web search | Tavily v2 with advanced params |
| Canvas | Fabric.js 6 |

## Getting Started

### 1. Clone and install

```bash
git clone https://github.com/aspershupadhyay/elite-mode-studio
cd elite-mode-studio
npm install
```

### 2. Set up Python backend

```bash
cd backend
python3 -m venv venv
source venv/bin/activate    # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# Edit .env — add your API keys
```

### 3. Get API keys (both have free tiers)

- **NVIDIA NIM** → [build.nvidia.com](https://build.nvidia.com)
- **Tavily** → [tavily.com](https://tavily.com)

### 4. Run

```bash
# Terminal 1 — Python backend (required first)
cd backend && python3 api.py

# Terminal 2 — Electron + React dev server
npm run dev
```
