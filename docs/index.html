How it works

From idea to post in 4 steps

It is really this simple. Even if you have never used AI tools before.

1

Download and set up

Download the app and paste in your free NVIDIA and Tavily API keys. Takes about 2 minutes.

2

Type your topic

Just describe what you want to post about. "5 tips for waking up early" or "news about AI this week" both work.

3

Pick your style

Choose your platform (Instagram, LinkedIn, etc.), tone (funny, professional, casual), and how many posts you want.

4

Generate and design

Click Generate. Your posts appear in seconds. Send them straight to the Design Studio to make graphics and carousels.

Mac users: getting a security warning?

macOS blocks apps without an Apple certificate. You can fix it in one of two ways:

Option 1 - Terminal (fastest)

Open Terminal and run this command, then double-click the app normally:

xattr -cr /Applications/CreatorOS.app

Option 2 - No terminal needed

Right-click the app in Finder, choose Open, then click Open in the popup. Done. It will work normally every time after that.

Setup

Two free API keys required

Both are free to get and take under 2 minutes to set up.

1

NVIDIA API Key

This powers all the AI writing. Go to build.nvidia.com, create a free account, and generate an API key. It starts with nvapi-

2

Tavily API Key

This lets the app search the web for fresh information. Go to app.tavily.com, sign up free, and copy your key. The free plan is more than enough.

3

Paste into the app

When you open CreatorOS for the first time it will ask for both keys. Paste them in and you are ready to go. You can update them anytime in Settings.
